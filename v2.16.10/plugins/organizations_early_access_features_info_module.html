cisco.meraki.organizations_early_access_features_info module – Information module for organizations _earlyaccess _features

Note

This module is part of the cisco.meraki collection (version 2.16.10).

To install it, use: ansible-galaxy collection install cisco.meraki. You need further requirements to be able to use this module, see Requirements for details.

To use it in a playbook, specify: cisco.meraki.organizations_early_access_features_info.

New in cisco.meraki 2.16.0

Synopsis

  • Get all organizations _earlyaccess _features.

  • List the available early access features for organization.

Note

This module has a corresponding action plugin.

Requirements

The below requirements are needed on the host that executes this module.

  • meraki >= 2.4.9

  • python >= 3.5

Notes

Note

  • SDK Method used are organizations.Organizations.get_organization_early_access_features,

  • Paths used are get /organizations/{organizationId}/earlyAccess/features,

  • Supports check_mode

  • The plugin runs on the control node and does not use any ansible connection plugins, but instead the embedded connection manager from Cisco meraki SDK

  • The parameters starting with meraki_ are used by the Cisco meraki Python SDK to establish the connection
